# User Story

As a user, I need to select a product and view dealers offering that product so that I can compare prices.

## Acceptance Criteria

Given the user opens the application  
When the page loads  
Then the product dropdown should be displayed  

Given the user selects a product  
When the product is selected  
Then the list of dealers supplying the product should be shown  

Given the user selects a dealer  
When the dealer is selected  
Then the price offered by that dealer should be displayed
