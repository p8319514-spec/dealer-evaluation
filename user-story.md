# User Story

## Title
Compare dealer prices for selected product

## User Story
As a user, I need to select a product and view dealers offering that product so that I can compare prices.

## Details and Assumptions
- Products are available in dropdown.
- Dealers are stored in database.
- Prices are shown for each dealer.
- User can compare prices.

## Acceptance Criteria

Given the user opens the website  
When the user selects a product  
Then dealers should be displayed

Given dealers are available  
When the user selects a dealer  
Then product price should be shown

Given multiple dealers exist  
When user compares prices  
Then best price can be chosen
